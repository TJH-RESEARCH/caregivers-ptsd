# Analysis code for Caregivers' Mental Health Survey. 

This project examines the mental health of caregivers, with a special interest in how military veterans differ from non-veterans. 

The replicable analysis is performed by the `do.R` file. For a walk through of the analysis, see the `notebook.html` file. Scripts called by the `do.R` file are stored in `R/`. The statistical model is implemented in Stan, stored in the `Stan/` folder.

The data are not publicly available.
